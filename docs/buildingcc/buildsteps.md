# Build steps
Follow these steps to build your system
1. Get Linux
2. Read the first third of Sobell's linux book
3. Install postgres
4. Install netbeans
5. Configure Java EE
6. Download and extract Glassfish 
7. Setup glassfish security
8. Build database
