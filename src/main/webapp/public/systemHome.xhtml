Release notes

  • Platinum Citations are now live for final testing. Viewing an old citation or creating a new citation on CECase profile now allows you to enter a docket number, record individual violation statuses, and create a date-stamped log of each phase of the citation life cycle. Photos/documents can be uploaded directly. Person links coming soon.
  • File uploads on CE case are working again using our unified photo-doc interface. File size limit is still around 10mb, but the solution has been found for an upcoming release.
  • Events now can contain person links! After you create an event, then click view/edit in its display table, and then "Manage event-person connections "
  • Mailing address linking is now active. You are always linking either a property or a person to the session's current address, which is displayed under the session Property session box. Manage property address connections in the property profile page and the person address connections in the person profile dialog, accessed with "person - view/edit" in the Person session box.
