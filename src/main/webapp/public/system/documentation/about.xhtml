About CodeNforce

CodeNForce is a municipal code enforcement and occupancy inspection permitting database system created by Technology Rediscovery LLC for the Turtle Creek Valley Council of Governments.

CodeNForce grew out of CMU's Heinz College culminating Systems project completed by students from both the Masters of Information Systems Management (MISM) program and the Masters in Public Policy (MPP) program. The union of the two fields created a robust project plan which was then implemented by one member of the project following graduation.
