Apply for an Occupancy Permit
If you would like to apply for an occupancy permit in your municipality, please begin using the process below.
3. Configure unit list

To apply for occupancy, please create a complete list of ALL units in the property, including those you do not want to apply for. To add a unit, click the "Add Unit" button. Click the checkmark under the "Edit" column to save your changes. Unit Number is required. Rental notes should describe your intentions to rent. The description should describe the room/unit (e.g. basement). Once finished, click the button for the next step.
