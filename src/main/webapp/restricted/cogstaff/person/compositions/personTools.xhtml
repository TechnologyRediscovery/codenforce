What is a multi-person?

A multi-person is a name that represents two or more people or a person and a corporate entity. This is common since we use exactly the name listed as a property owner by Allegheny County as the person name, and many folks jointly own properties in the county.

For example, a property might be owned by "SHELLY MARY R AND PERCY B". We would enter this entire String of characters as the Person name in codeNforce.

In other words, we don't try to parse the name into separate person records. Instead, we turn on this flag to note that this Person represents two or more people in the world or a person and a business.
