Add a code element to a municipality-specific code set to make them enforcable, meaning they can be attached to cases, violations, and citations.

Add Code Elements by Source

Select a code source (e.g. IPMC 2015) from which you wish to add a code element to this set.

Select multiple elements by clicking them sequentially. Unselect a row by holding down control while clicking (using a mouse for this is handy).

The values in these input boxes will be copied into each of the code elements you choose to add to the code set. This allows for bulk adding of code elements all with the same enforcability data. Values for each individual element can always be edited by loading the assembled code set through the "Code Sets" page link in the left side bar.
