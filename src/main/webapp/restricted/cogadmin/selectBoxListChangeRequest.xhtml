Select item list change request

To expedite development of our core system, the following selectable lists are 100% customizable, but must be edited "in the back-end" by the administrator. This allows us to prioritize developing more frequently used pages in the system. Eventually, each select item list will have a GUI go to with it's table.

List of selectable lists without GUI (Graphic User Interface)

  1. Code enforcement action request issue type (table name: actionrequestissuetype)
  2. Code enforcement action request status (table name: ceactionrequeststatus)
  3. Code enforcement citation status (table name: citationstatus)
  4. Code enforcement letter text block category (table name: textblockcategory)
  5. Reason for submitting an occupancy permit application (table name: occpermitapplicationreason)
  6. Occupancy inspection status (table name: occupancyinspectionstatus)
  7. System improvement type (table name: improvementtype)
  8. Improvement suggestions status (table name: improvementstatus)
