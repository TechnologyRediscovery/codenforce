Notes for : Tuesday, 7 Aug 2018

Recent system updates

  1. Event editing (which somehow didn't get made before) is up and running
  2. Timeline event notices now can be viewed and cleared from the Dashboard. Users can specify the date range to display
  3. Our server's memory usage was running at about 80% during off-hours. This is too high. With a few clicks, this was upgraded from 4GB to 8GB.

Migration Notes

Migration from Access was a decent success; my informal estimate from work with Kyle is about 80% of data made it in without error. Inconsistencies in dates entered into Access were magnified during the migration, and some properties with data in Access don't contain any data. Please consider creating a new case for properties in which legacy data is too inaccurate to provide useful future reference.
